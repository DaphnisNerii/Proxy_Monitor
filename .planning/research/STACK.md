# Research: UI & Charting Stack

## UI Frameworks Comparison

| Framework | Aesthetics | Ease of Use | Performance | Bundle Size |
|-----------|------------|-------------|-------------|-------------|
| **Flet** | ★★★★★ | ★★★★★ | ★★★★☆ | ~20MB |
| **PySide6** | ★★★★☆ | ★★★☆☆ | ★★★★★ | ~60MB+ |
| **CustomTkinter** | ★★★☆☆ | ★★★★★ | ★★★☆☆ | <1MB (extra) |

### Recommendation: Flet
For a "Rich Aesthetics" goal, Flet is the winner. It allows building professional-grade UIs that look like modern mobile/web apps with very little code.

## Charting Libraries

| Library | Type | Speed | Beauty |
|---------|------|-------|--------|
| **Flet Charts** | Built-in | Fast | Excellent |
| **PyQtGraph** | Qt-based | Blazing | Industrial |
| **Matplotlib** | Generic | Moderate | Generic |

### Recommendation: Flet Bar/Line Charts
Integeated perfectly with Flet UI, supports smooth animations.
